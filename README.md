# Poke TUI (Go)

A Go-based Text User Interface (TUI) application using the PokeAPI to display Pokémon and their attributes.

## Requirements

- Go 1.18+

Install dependencies:
```bash
go mod tidy
```

## Usage

```bash
go run main.go
```

Within the TUI:
- Arrow keys to navigate.
- Enter to select a Pokémon and view details.
- Press `q` to quit.
